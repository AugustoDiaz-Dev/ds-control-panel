Project Overview

This project creates an interactive Data Science control panel for real-time experimentation using Jupyter notebooks and a FastAPI backend. It focuses on a financial classification problem and compares multiple algorithms including LightGBM, XGBoost, Random Forest, and others.

Quick Start

1. Train Models

POST http://localhost:8000/train?n_estimators=100&learning_rate=0.1&max_depth=5

2. Make Predictions

POST http://localhost:8000/predict Content-Type: application/json { "features": [750, 80000, 15000, 35, 60000, 8, 25000] }

3. View Metrics

GET http://localhost:8000/metrics

Dataset

The project uses a synthetic financial dataset with the following features:

  • credit_score: Credit score (normalized)
  • income: Annual income
  • debt: Current debt
  • age: Age of applicant
  • loan_amount: Requested loan amount
  • employment_years: Years of employment
  • savings: Current savings

Target: Binary classification (1 = Good credit risk, 0 = Bad credit risk)
